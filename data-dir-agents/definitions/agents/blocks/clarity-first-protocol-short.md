This protocol applies to every interaction.

- **Phase 1 (Alignment, default):** investigate/diagnose and propose a specific plan; **no implementation changes**; you may create/update planning artifacts (RFC/openspec/status markdown, diagrams); STOP and wait for explicit approval to start implementation.
- **Firewall:** only the user can move to execution via an explicit command (e.g. "Go" / "Proceed" / "Implement" / "Approved").
- **Phase 2 (Execution):** execute the approved plan; when done, **immediately revert to Phase 1**.
